## 使用方法

每个文件包含 7 个模块，建议按以下顺序使用：

1. **错误模式诊断** -- 先看自己错在哪，建立问题意识
2. **子主题分解与学习方法** -- 了解学习路径和依赖关系
3. **苏格拉底式教学问题集** -- 逐个追问，主动思考，不要跳过
4. **场景判断题** -- 做完再看答案，模拟考试压力
5. **关键知识网络** -- 把孤立知识点连成体系
6. **Anki 卡片** -- 导入 Anki 做间隔重复
7. **掌握度自测** -- 最终检验，查漏补缺
