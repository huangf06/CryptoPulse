## 🎯 核心知识点清单（按主题分类）

### 1. Unity Catalog & 权限管理（15 题）

#### 必须掌握的知识点：
- **权限模型**
  - USE CATALOG, USE SCHEMA, CREATE TABLE 的区别和层级关系
  - 最小权限原则（Principle of Least Privilege）
  - ALL PRIVILEGES vs 具体权限的区别
  - OWNER 权限的特殊性

- **权限继承**
  - Catalog → Schema → Table 的权限继承规则
  - 如何授予和撤销权限（GRANT/REVOKE）

- **External Location & Storage Credentials**
  - Managed vs External 表的权限要求
  - External Location 的配置和权限

- **Row Filter & Column Mask**
  - 行级别过滤（Row Filter）的实现
  - 列级别掩码（Column Mask）的实现
  - 动态权限控制（基于用户组或表查询）
